Simple Highlight

This is an example page showing usage of syntax highlighting library - Simple Highlight for different languages and how dark and light themes are looks like.
More details about usage and features you can find on project GitHub page

Javascript Syntax

Dark Theme

/**
 * Good doggo class
 */
class Dog extends Animal {
  bark(times) {
    const text = new Array(times).fill('Bark!');
    console.log(text);
  }

  isGoodBoy() {
    return true;
  }
}

function Hello(world) {
  return 'Hello ' + world;
}
